# My OpenClaw Skills

由 AI Agent（幕僚长）创建和维护的技能集合，用于自动化处理日常业务任务。

## 已包含技能

### 📊 delivery-plan-stats — 发货计划统计（v2.0）

自动处理销售单数据，生成未发货订单透视表和统计表。

**功能：**
- 自动判断客审状态（标记列含"客审"/"货审"→已客审，否则→未客审）
- 识别发货时间类型（加急/指定时间→指定时间；有货就发/等通知→等通知）
- 生成状态合并列和统计项目列
- 按销售渠道透视，统计表按合计金额降序排列
- 输出Excel（含处理后数据、透视表、统计表三个sheet）和统计表截图

**使用：** 发送原始销售单Excel文件，说"统计发货计划"即可触发

**依赖：** Python 3.8+, pandas, openpyxl, matplotlib

---

### 📥 april-delivery-data — 4月发货数据获取

从WPS在线文档读取4月发货汇总金额。

**功能：**
- 自动从WPS文档"2604发货"子表E43单元格读取汇总金额
- 返回金额数据

**使用：** 说"给我拉下4月发货数据"即可触发

**依赖：** kdocs-cli 2.4.3+

## 在线工具

### 🌐 [未发货订单统计工具 v3.0](https://seaglet.github.io/My-Openclaw-Skills/未发货订单统计工具.html)

纯前端网页版，拖入Excel文件即可一键生成统计结果，支持：
- URL输入或文件拖拽上传
- 统计表预览和图表展示
- Excel和图片下载
- 手机端长按图片保存
- 月内发货计划金额统计柱状图（v3.0新增）
- 年月选择，按日统计发货金额（v3.0新增）
- 历史记录保存，下拉选择回看（v2.0新增）
- 图片Data URL格式，支持长按保存（v2.0新增）

## 技能结构

每个技能遵循统一的目录结构：

```
skill-name/
├── SKILL.md          # 技能说明文档
├── scripts/          # 执行脚本
└── references/       # 参考文档（可选）
```
