# My OpenClaw Skills

个人创建的 OpenClaw 技能集合。

## 技能列表

### 1. delivery-plan-stats（发货计划统计）

**触发词：** 统计发货计划、发货计划统计

**功能：** 自动处理销售单数据，生成发货计划透视表

**输入：** Excel文件（含"销售单"子表）

**输出：**
- 处理后的Excel文件（含透视表）
- 透视表图片

**处理逻辑：**
1. 插入"客审？"列（判断标记列）
2. 插入"发货时间类型"列
3. 插入"发货订单类型"列
4. 插入"发货？"列
5. 创建透视表（行：销售渠道，列：发货状态，值：应收合计）

---

### 2. april-delivery-data（获取4月发货数据）

**触发词：** 给我拉下4月发货数据、获取4月发货数据

**功能：** 从WPS在线文档获取4月发货汇总金额

**数据源：** WPS在线文档"2604发货"子表E43单元格

**输出：**
- 原始值（带千分位分隔符）
- 万元值

**技术特点：**
- 使用持久化session保持登录状态
- 通过JS API直接读取单元格（0.17秒）
- 只读操作，不修改文档

---

## 使用方式

将技能文件夹放到 `.agents/skills/` 目录下，Agent会自动识别加载。
